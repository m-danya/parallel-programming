Для генерации команд компиляции и отправки задач в очередь были написаны скрипты
на языке Python 3: `generate_submit.py`. Пример вывода, т.е. сами команды для
компиляции и постановки задач в очередь приложены в виде файлов `compile.sh` и
`submit_1.sh`, `submit_2.sh`, `submit_3.sh`.

Результаты были агрегированы из выходных файлов автоматически с помощью
Python-скрипта `parse_output.py`.
